# v2 Reference

`apps/game/v2/` bleibt im Repo als Referenz fuer aeltere Three.js-/DOM-Experimente, Flows und Content-Ideen.

## Wichtig
- Nicht Teil des kanonischen Produkt-Builds
- Nicht Teil von `apps/game/dist`
- Nicht die oeffentliche Runtime auf `games.menschlichkeit-oesterreich.at`

Die aktive Produktlaufzeit liegt im Root von `apps/game/` und basiert auf Babylon.js.
