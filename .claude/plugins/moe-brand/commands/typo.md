# Typografie Menschlichkeit Österreich

## Schriften

- **Headlines**: Nunito Sans (ExtraBold 800, Bold 700, SemiBold 600)
- **Fließtext**: Source Sans 3 (Regular 400, SemiBold 600, Bold 700)
- **Zitate**: Merriweather Italic (NUR Blockzitate)

## Hierarchie

| Element | Schrift       | Gewicht | Desktop | Mobil | Zeilenhöhe |
| ------- | ------------- | ------- | ------- | ----- | ---------- |
| H1      | Nunito Sans   | 800     | 48px    | 36px  | 1.2        |
| H2      | Nunito Sans   | 700     | 36px    | 28px  | 1.25       |
| H3      | Nunito Sans   | 700     | 28px    | 24px  | 1.3        |
| H4      | Source Sans 3 | 600     | 22px    | 20px  | 1.35       |
| Body    | Source Sans 3 | 400     | 16px    | 16px  | 1.6        |
| Button  | Nunito Sans   | 700     | 16px    | 16px  | 1.2        |

## Regeln

- IMMER linksbündig, NIEMALS Blocksatz
- Zeilenlänge: 60-80 Zeichen
- Mindestgröße: 16px Web, 10pt Print
- Bold für Hervorhebungen, Unterstrichen NUR für Links

## CSS

```css
font-family: 'Source Sans 3', 'Segoe UI', Roboto, Arial, sans-serif;
font-family: 'Nunito Sans', 'Segoe UI', Roboto, Arial, sans-serif;
```
