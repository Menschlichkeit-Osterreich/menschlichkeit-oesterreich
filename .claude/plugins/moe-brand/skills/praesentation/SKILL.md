# Präsentationen – Menschlichkeit Österreich

Erstelle Folienpräsentationen nach den Brand Guidelines.

## Masterfolien (3 Typen)

### 1. Titelfolie

- Hintergrund: Demokratie-Blau (#1B4965)
- Text: Weiß
- Logo: Links unten
- Titel: Nunito Sans ExtraBold 60pt
- Untertitel: Source Sans 3 SemiBold 32pt

### 2. Inhaltsfolie

- Hintergrund: Weiß (#FFFFFF)
- Farbiger Akzentstreifen links oder oben (4-6px, Logo-Orange oder Blau)
- Logo: Rechts unten (klein)
- Titel: Nunito Sans Bold 40pt, Demokratie-Blau
- Body: Source Sans 3 Regular 24pt, Neutral 700

### 3. Schlussfolie / CTA

- Hintergrund: Text-Orange (#B54A0F)
- Text: Weiß
- CTA prominent
- Kontaktdaten
- Logo: Zentriert

## Typografie auf Folien

| Element           | Schrift       | Gewicht       | Größe |
| ----------------- | ------------- | ------------- | ----- |
| Folien-Titel      | Nunito Sans   | ExtraBold 800 | 60pt  |
| Folien-Untertitel | Source Sans 3 | SemiBold 600  | 32pt  |
| Folien-Body       | Source Sans 3 | Regular 400   | 24pt  |
| Folien-Caption    | Source Sans 3 | Regular 400   | 16pt  |

## Layout-Regeln

- Seitenverhältnis: 16:9 (1920×1080px)
- Maximal 1 Kernaussage pro Folie
- Maximal 5-6 Textzeilen pro Folie
- Mindestens 30 % Weißraum
- Bilder: Vollflächig oder 60 % der Folie
- Icons: Phosphor Icons, 48px auf Folien
- Keine Clipart, keine WordArt, keine Schatten

## Farblogik Folien

- Titelfolie: Demokratie-Blau
- Inhalt: Weiß + Blau-Akzent
- Daten/Zahlen: Kennzahlenkarten mit Orange-Zahlen
- Zitate: Neutral 100 Hintergrund, Merriweather Italic
- CTA/Schluss: Text-Orange

## Diagramme auf Folien

Serie 1: #1B4965, Serie 2: #B54A0F, Serie 3: #00695C, Serie 4: #8B6F4E
Keine 3D, keine Schatten, direkte Beschriftung bevorzugen.

## Wenn $ARGUMENTS ein Thema enthält

1. Erstelle eine Foliengliederung (10-15 Folien)
2. Bestimme für jede Folie den Masterfolien-Typ
3. Generiere Textinhalte im Brand Voice
4. Gib PPTX (via python-pptx) oder HTML-Slides aus
