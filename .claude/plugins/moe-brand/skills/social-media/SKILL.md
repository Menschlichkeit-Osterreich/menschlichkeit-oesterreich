# Social-Media-Templates – Menschlichkeit Österreich

Erstelle Social-Media-Grafiken streng nach den Brand Guidelines.

## Formate & Maße

| Typ               | Maße        | Seitenverhältnis |
| ----------------- | ----------- | ---------------- |
| Instagram Feed    | 1080×1350px | 4:5              |
| Instagram Story   | 1080×1920px | 9:16             |
| LinkedIn Post     | 1200×1200px | 1:1              |
| Facebook Cover    | 851×315px   | ~2.7:1           |
| LinkedIn Banner   | 1584×396px  | 4:1              |
| Karussell (Slide) | 1080×1350px | 4:5              |

## Vier Farbkategorien für Feed-Kohärenz

| Kategorie | Hintergrund                     | Einsatz                      |
| --------- | ------------------------------- | ---------------------------- |
| Mission   | Demokratie-Blau #1B4965         | Werte, Demokratie, Rechte    |
| Mitmachen | Text-Orange #B54A0F             | CTAs, Aktionen, Events       |
| Wissen    | Solidaritäts-Petrol #00695C     | Fakten, Bildung, Statistiken |
| Community | Menschlichkeits-Warmton #8B6F4E | Zitate, Dank, Gemeinschaft   |

## Layout-Regeln für jeden Post

1. **Logo**: Unten rechts, 10-12 % der Kartenfläche
2. **Headline**: Obere Hälfte, Nunito Sans ExtraBold, mindestens 48pt auf 1080px Breite
3. **Unterstützendes Element**: Bild, Icon oder Farbfläche
4. **Optionaler Footer-Streifen**: Website-URL

## Typografie auf Social Media

| Element  | Schrift       | Gewicht       | Mindestgröße |
| -------- | ------------- | ------------- | ------------ |
| Headline | Nunito Sans   | ExtraBold 800 | 48pt         |
| Subline  | Source Sans 3 | SemiBold 600  | 28pt         |
| Body     | Source Sans 3 | Regular 400   | 24pt         |

## Zitatgrafik

- Hintergrund: Neutral 100 (#F0EBE6) oder Demokratie-Blau
- Zitat: Merriweather Italic
- Personenname: Source Sans 3 SemiBold
- Logo: Klein in der Ecke
- NUR echte Zitate von echten Menschen

## Karussell (10 Slides)

- Slide 1: Hook (starke Headline + Bild)
- Slides 2-9: Inhalt (ein Punkt pro Slide)
- Slide 10: CTA + Logo
- Konsistentes Layout über alle Slides

## Story (9:16)

- Text im sicheren Bereich (obere und untere 10 % frei lassen)
- Kurze, prägnante Texte (max. 2-3 Zeilen)
- Vertikale Orientierung beachten

## Spendenaufruf

- Emotionales Bild oder Icon
- Konkrete Wirkung: "Deine 30 € ermöglichen..."
- CTA-Button prominent
- NIEMALS Mitleid erzeugen oder Druck aufbauen

## Template-Ausgabe

Wenn $ARGUMENTS ein Thema enthält, erstelle:

1. SVG/HTML-Code für die Grafik (1080×1350px Standard)
2. Passende Farbkategorie automatisch wählen
3. Textvorschlag für den Post-Text (max. 200 Zeichen)
4. 3-5 passende Hashtags

Verwende immer:

- Google Fonts: Nunito Sans + Source Sans 3
- Phosphor Icons statt Emojis
- Weißen Text auf Farbhintergründen (Kontrast prüfen!)
