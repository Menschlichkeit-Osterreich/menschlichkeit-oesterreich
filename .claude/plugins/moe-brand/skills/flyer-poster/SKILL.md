# Flyer & Poster – Menschlichkeit Österreich

Erstelle Print-Material nach den Brand Guidelines.

## Flyer DIN lang (210×105mm)

### Vorderseite

- Oben: Headline (Nunito Sans ExtraBold, min. 24pt)
- Mitte: Bild oder Grafik
- Unten: CTA + Logo
- Hintergrund: Weiß oder Farbfläche (Blau/Orange)

### Rückseite

- Informationstext (Source Sans 3 Regular, min. 10pt)
- Kontaktdaten
- QR-Code zur Website
- Logo klein unten

### Papier: 135-170g/m², Recycling (Blauer Engel oder FSC Recycled)

## Poster DIN A3 (297×420mm) / A2

### Layout-Drittel

- Oberes Drittel: Headline
- Mittleres Drittel: Bild oder Grafik
- Unteres Drittel: Informationen + Logo + CTA

### Typografie Print

| Element  | Schrift               | Mindestgröße         |
| -------- | --------------------- | -------------------- |
| Headline | Nunito Sans ExtraBold | 36pt (A3), 48pt (A2) |
| Body     | Source Sans 3 Regular | 14pt (A3), 16pt (A2) |
| Caption  | Source Sans 3 Regular | 10pt                 |

### Mindestens 30 % Weißraum

## Roll-Up-Banner (850×2000mm)

- Leserichtung: Von unten nach oben
- Logo + Kontakt: Unten (Augenhöhe)
- Hauptbotschaft: Mitte
- Headline: Oben (min. 80pt)
- Body: Min. 24pt

## Visitenkarte (85×55mm)

### Vorderseite

- Logo, Name, Position
- Kontaktdaten (Telefon, E-Mail, Website)

### Rückseite

- Leitwerte oder QR-Code zur Website
- Papier: 300-400g/m², ungestrichen, Recyclingpapier

## Allgemeine Print-Regeln

- Farbmodus: CMYK
- Beschnittzugabe: 3mm allseitig
- Schriften einbetten oder in Pfade wandeln
- Bildauflösung: min. 300 DPI
- Logo-Orange CMYK: 0, 75, 90, 15
- Demokratie-Blau CMYK: 90, 45, 20, 40
- Text nie kleiner als 10pt (Print)
- Nachhaltiges Papier als Default

Wenn $ARGUMENTS ein Event oder Thema enthält, erstelle eine druckfertige Vorlage als HTML/SVG mit korrekten Maßen.
